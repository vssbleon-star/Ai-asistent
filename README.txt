 Danil AI - Десктопный ассистент

Простое десктопное приложение AI-ассистента с графическим интерфейсом на PyQt6.

 Возможности

-  Общение с AI: Использует OpenAI API или локальную модель через Ollama
-  История диалогов: Сохраняет все разговоры в базе данных SQLite
-  Управление диалогами: Создание, переключение и удаление диалогов
-  Гибкие настройки: Возможность настройки API ключей
-  Удобный интерфейс: Современный дизайн с разделением на панели

 Установка

 Требования
- Python 3.8+
- PyQt6
- requests

 Установка зависимостей
```bash
pip install PyQt6 requests
```

 Для использования локальной модели
1. Установите [Ollama](https://ollama.ai/)
2. Загрузите модель (например, llama2):
```bash
ollama pull llama2
```

 Использование

Запуск приложения
```bash
python assistant.py
```

 Интерфейс
- Левая панель: Список диалогов с кнопками создания/удаления
- Основная область: История сообщений текущего диалога
- Панель ввода: Поле для ввода сообщений с кнопкой отправки
- Панель инструментов: Выбор модели и очистка диалога

 Горячие клавиши
- `Ctrl+Enter` - Отправить сообщение
- `Ctrl+N` - Создать новый диалог

 Настройка

 API ключи
1. Откройте Настройки (кнопка в левой панели)
2. Введите ваш OpenAI API ключ (начинается с `sk-`)
3. Или оставьте поле пустым для использования локального режима

Модели
- Локальный режим: Llama 2 через Ollama
- OpenAI API: GPT-3.5 Turbo / GPT-4

Структура базы данных

Приложение использует SQLite базу данных `chat_history.db`:
- `conversations` - Диалоги (ID, название, дата создания)
- `messages` - Сообщения (ID диалога, роль, содержание, время)
- `settings` - Настройки приложения

Требования

Для локального режима
- Установленный Ollama
- Достаточно памяти для запуска выбранной модели

Для онлайн режима
- Действительный API ключ OpenAI
- Доступ к интернету

Примечания

- Все сообщения сохраняются автоматически
- Можно очищать отдельные диалоги без их удаления
- При первом запуске автоматически создается новый диалог
- Поддерживается кэширование сообщений для быстрой работы
